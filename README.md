# Friend_Recommendation_System
An interactive social networking application built with React and Graph Theory principles. This system suggests potential friends to users by analyzing mutual connections (common neighbors) and provides a simulated AI-driven icebreaker feature to help initiate conversations.
Graph-Based Recommendations: Suggests users based on the "Common Neighbors" algorithm.

Dynamic Network Management:

Register new users in the network.

Establish mutual friendships between existing users.

Real-time Ranking: Recommendations are automatically sorted in descending order of mutual friend count.

Network Insights: View a detailed list of current friends and users not yet connected.

Data Persistence: Uses LocalStorage to save your network state (users and friendships) so data is preserved across browser sessions.

Simulated AI Icebreakers: A dedicated feature to generate conversation starters based on shared social context.

Responsive UI: A modern, tabbed interface built with CSS Modules and Lucide-React icons.
